# A Template for React Native App to start your cross platform app development

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Quick Start
1. Clone this repository
```bash
git clone https://github.com/yousuf-git/react-native-first-app.git
```
2. Install dependencies
```bash
npm install
```
3. Start the app
```bash
npx expo start
```

Congratulations! You have successfully created a React Native app using Expo, with Nativewind already configured !
